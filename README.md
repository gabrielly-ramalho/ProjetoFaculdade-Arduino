# ProjetoFaculdade-Arduino
Projeto de trabalho acadêmico de Arduíno

# O DESAFIO:
Foi solicitado um projeto em arduíno que contasse com pelo menos dois sensores

# O PROJETO:
Minha ideia foi desenvolver um detector de incêndio.
O detector tem um sensor de fumaça e um sensor de temperatura:
 - O sensor de fumaça aciona o alarme quando detecta fumaça e liga a luz vermelha;
 - O sensor de temperatura quando detecta temperaturas de 35°C ou mais aciona as luzes de alerta;
 - Quando não há nenhum alerta a luz verde segue ligada indicando que está tudo certo.
